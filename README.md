- Sinh viên thực hiện: Nguyễn Bảo Trung 
- Lớp: DK24TTC5

## Giới thiệu

Đây là đồ án môn học với đề tài xây dựng website bán linh kiện điện tử sử dụng ASP.NET Core MVC.  
Website cho phép người dùng xem sản phẩm, tìm kiếm, thêm vào giỏ hàng, thêm vào danh sách yêu thích, đặt hàng và quản lý thông tin cá nhân.

Hệ thống được xây dựng theo mô hình MVC, sử dụng Entity Framework Core và SQL Server để quản lý dữ liệu.

---

## Công nghệ sử dụng

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- HTML / CSS / JavaScript
- Visual Studio 2022

## Chức năng đã thực hiện

### Người dùng

- Xem danh sách sản phẩm
- Lọc sản phẩm theo danh mục
- Tìm kiếm sản phẩm
- Xem chi tiết sản phẩm
- Thêm vào giỏ hàng
- Danh sách yêu thích (Wishlist)
- Đăng ký / đăng nhập
- Quản lý thông tin cá nhân
- Đổi mật khẩu
- Đặt hàng
- Xem lịch sử đơn hàng

### Giỏ hàng

- Thêm sản phẩm vào giỏ
- Cập nhật số lượng
- Xóa sản phẩm
- Hiển thị tổng tiền
- Hiển thị số lượng trên icon

### Wishlist

- Thêm sản phẩm vào danh sách yêu thích
- Xóa khỏi wishlist
- Hiển thị số lượng wishlist trên layout

### Đơn hàng

- Tạo đơn hàng
- Lưu Order + OrderDetail
- Xem danh sách đơn hàng
- Xem chi tiết đơn hàng
- Trạng thái đơn hàng

### Danh mục

- Hiển thị danh mục sản phẩm
- Lọc sản phẩm theo danh mục
